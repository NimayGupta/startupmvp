# Shopify App Scaffold Instructions

This directory is where the Shopify CLI Remix app lives.
The core files are generated by the CLI and must be created before the extension
files in this repo take effect.

## Step 1: Run the CLI scaffold

From the **repository root** (not inside app/):

```bash
npx @shopify/cli app init \
  --name "discount-optimizer" \
  --template remix \
  --path ./app \
  --package-manager npm
```

The CLI will open a browser window to authenticate with your Shopify Partner account.
Follow the prompts. When complete, the following files will be generated inside `app/`:

- `package.json` (with @shopify/shopify-app-remix, @remix-run/react, @shopify/polaris)
- `shopify.app.toml` (CLI manifest)
- `app/shopify.server.ts` (auth/session middleware — **extend, don't replace**)
- `app/routes/auth.$.tsx` (OAuth callback — **keep as generated**)
- `app/routes/app.tsx` (Polaris AppProvider shell)
- `app/routes/_index.tsx` (redirect to /app)
- `app/routes/webhooks.tsx` (stub — **replaced by our version**)

## Step 2: After scaffold, update shopify.app.toml

Replace the generated `shopify.app.toml` with the one in this repo
(`app/shopify.app.toml`), substituting your actual `SHOPIFY_API_KEY`.

## Step 3: Extend shopify.server.ts

After the CLI generates `app/shopify.server.ts`, apply the database session
storage patch from `app/lib/db-session-storage.server.ts`.

## Step 4: Verify

```bash
cd app
npm install
shopify app dev
```

The app should open in your development store at the tunnel URL.
